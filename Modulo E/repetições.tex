repetição teste logico no inicio:

while (condição){

}

obs: enquanto (while) a condição foi verdadeira o bloco irá acontecer.

exemplo: 
contador=1
while (contador<=10){
    console.log(`passo ${contador}`)
    contador++
}

repetição teste logico no final :

do{

} while ()

exemplo: 
contador=1
do{
    console.log(`passo ${contador}`)
    contador++
} while (contador<=10)

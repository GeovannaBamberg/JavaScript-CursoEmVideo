variaveis compostas são capazes de armazenar varios valores em uma mesma estrutura

exemplo variavel simples vs variaveis compostas:
var a1 = "carro1"
var a2 = "carro2"
var a3 = "carro3"

vs 

var vagasAutomoveis = ["carro1", "carro2", "carro3"]

console.log vagasAutomoveis[0]

explicando:
vagasAutomoveis no programa acima é um Array ou vetor
O vetor é composto de elementos (["carro1", "carro2", "carro3"]), que são guardados por indice, e valor de conteudo.

um array é uma variavel que tem varios elementos cada elemento tem um valor uma chave de indentificação.
